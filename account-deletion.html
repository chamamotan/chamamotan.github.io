Account Deletion

Request deletion of your account

This page explains how to request deletion of your account and associated data for mobile games published by PLUMERIA Inc.

If you signed in to a game with your Google account, send an email to contact@plumeriagames.com from that Google account's email address, with the game name in the subject line. The steps below explain the details.

How to request deletion

If you signed in with Google

  1. Compose an email to contact@plumeriagames.com from the email address of the Google account you use to sign in to the game.
  2. Put the game name in the subject line, using this format: [Game Name] Account Deletion Request — for example, [Come On Zombie] Account Deletion Request. The exact subject format for each game is listed in the game sections below.
  3. Optionally, include your in-game user ID (UUID) in the email body. This is not required — it is only useful if you have more than one account in the same game and want to specify which one to delete.
  4. We will verify that the sender address matches an account in the game and delete the account and its data. Processing takes up to 30 days. We will reply to confirm when deletion is complete.

The request must be sent from your Google account's email address. The sender address is how we verify that the request comes from the account owner. Requests sent from any other address cannot be verified and cannot be processed.

The subject line must include the game name. contact@plumeriagames.com is a shared address for all PLUMERIA games, and each game stores its accounts separately. Without the game name we cannot locate your account.

Account deletion is permanent and cannot be undone.

If you play as a guest

Guest accounts are anonymous and have no email address attached, so there is no way for us to verify ownership of a guest account through this page or by email. To delete a guest account, use the account deletion option in the game's in-app settings screen.

If you have already uninstalled the app, please note that we have no way to identify which guest account was yours, so we cannot delete it on request. Reinstalling the app and signing in as a guest creates a new account and does not restore access to the previous one.

Games

Each section below lists, for one game, the email subject line to use and the data that is deleted or retained when the account is deleted.

Come On Zombie

Email subject line

[Come On Zombie] Account Deletion Request

Data that is deleted

  • Account information (personal data, stored in Firebase Authentication): email address and user ID.
  • Game progress data (stored in Firebase Realtime Database): gold, gems, energy, account experience, summon tickets (normal and premium), owned characters with their levels and awakening status, deck composition, stage progress, counterattack mode data, and quest and achievement progress.

Data that is retained

Analytics data collected by Firebase Analytics is not tied to your game account and is not removed by account deletion. It expires and is deleted automatically after 14 months, in line with the data retention settings of this game's Firebase project.
